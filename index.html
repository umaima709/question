Q. Is variable is case sensitive ?

JavaScript is a case-sensitive language. This means that language keywords, variables, function names, and any other identifiers must always be typed with a consistent capitalization of letters. The while keyword, for example, must be typed “while”, not “While”

Q. When we declare a variable can we start it with any number?

A variable name must start with a letter or the underscore character. A variable name cannot start with a number.
